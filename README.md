# ONIVERSE Project — Interview Tier 2

## Getting started

### Prerequisites

- Node.js v22
- pnpm v10.8+

This project uses [Mise](https://github.com/jdx/mise) to manage tool versions.

### Install

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Create a `.env` file

   ```bash
   cp .env.sample .env
   ```

3. Start the development server

   ```bash
   pnpm dev
   ```
